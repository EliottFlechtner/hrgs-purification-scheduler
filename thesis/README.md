# Internship Report (LaTeX)

Research Internship Report for the Double Degree Programme — EPITA & Institut
Polytechnique de Paris, written during the internship at the AQUA lab, Keio SFC.

## Building

```bash
cd thesis
make          # runs latexmk -pdf -interaction=nonstopmode main.tex
```

Requires a TeX Live installation with `latexmk`, `biber`, and the standard
`geometry`, `fancyhdr`, `biblatex`, `algorithm2e`, `listings`, and `glossaries`
packages. The `cairosvg` Python package is needed only if regenerating logo PNGs
from SVG source (see `images/`).

Output: `thesis/main.pdf`

## Directory structure

```
thesis/
├── main.tex               ← root document (includes all chapters)
├── latexmkrc              ← latexmk configuration
├── Makefile               ← build wrapper
├── styles/
│   ├── packages.tex       ← \usepackage declarations
│   ├── formatting.tex     ← fancyhdr headers/footers, hyperref, colors
│   ├── commands.tex       ← custom macros (\TODO, etc.)
│   └── theorem.tex        ← theorem/definition environments
├── frontmatter/
│   ├── titlepage.tex      ← cover page with institutional logos
│   ├── abstract.tex
│   ├── acknowledgements.tex
│   └── dedication.tex
├── chapters/              ← one .tex file per chapter (01–08 + appendix)
├── sections/              ← acronyms, notation, symbols
├── tables/                ← standalone table files
├── figures/               ← figures organised by chapter
├── images/                ← logo PNGs for the title page
└── bibliography/
    ├── references.bib     ← cited works
    └── mypapers.bib       ← own publications
```

## Relation to the codebase

The thesis documents the research performed using the scheduler/optimizer in
`src/hrgs_scheduler/`. Figures are generated by the experiment scripts in
`experiments/` and placed under `thesis/figures/`. The thesis directory is
intentionally self-contained: building it requires only a TeX distribution,
not the Python package.
